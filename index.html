      Cocktails

      Cocktials é uma aplicação Web desenvolvida em React, criada com o intuito de ajudar aqueles com menos experiência a fazerem os seus próprios cocktials em casa. Os utilizadores podem pesquisar por nomes, ou então colocar filtros como, o tipo de copo, nível de álcool, tipo de bebida, ingredientes, etc. A aplicação vai buscar toda a informação a uma API destinada a cocktails, para colmatar a informação, quando um cocktail não tem um video de demonstração associado na API, a aplicação recorre à API de dados do Youtube para apresentar três videos associados ao nome do cocktail pesquisado. Por último, recorri ao Google Firestore para proceder à autenticação dos utilizadores e armazenar dados, permitindo aos utilizadores de guardarem os seus melhores cocktails numa página de favoritos. É dado ao utilizador também um feedback de ações, como adicionar e remover dos favoritos, erros de pedido à API e erros de inicio de sessão.

      Cosmozone

      Num contexto de CBL (Challenge Based Learning), em parceria com o Planetário do Porto, eu e a minha equipa desenvolvemos uma aplicação com o intuito de promover eventos de todos os CCVs (Centros de Ciência Viva) do país. Assim cada CCV pode publicar um evento, o qual está a organizar, sendo permitido aos utilizadores que observem os detalhes do evento, demonstrem o seu nível de interesse na participação do mesmo, partilhar/trocar informações (texto/imagens/video) num fórum dedicado ao evento. Após o evento acontecer, é possível classificá-lo. Desenvolvemos também um back-office para os administradores (CCVs), acederem às estatísticas dos seus eventos, gerirem a base de dados e todos os eventos e informações associadas e garantirem a funcionalidade da aplicação.

      Top Indian Movies

      Top Indian Movies, foi um projeto individual, com o objetivo de conceber um site seguro para listar e filtrar filmes indianos a partir de uma base de dados fornecida. O site inclui mecanismos de registo e login de utilizadores com hashing de palavras-passe para segurança. Foram tomadas medidas para evitar injecções de SQL e outros potenciais riscos de segurança. Este projeto melhorou a minha compreensão da organização do código, assim como todos os ficheiros o nomes dos mesmos.
